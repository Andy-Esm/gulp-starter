# Сборка Gulp для верстки v1.0

Сборка предназначена в первую очередь для ускорения и автоматизации при верстке,
включая автоматическую простановку префиксов, оптимизацию изображений, компиляцию scss кода в минифицированый css файл.

В папке SCSS выполнена предварительная разбивка на блоки, подключен normalize.css, созданы переменные для цвета и шрифтов, созданы
миксины с наиболее часто используемыми свойствами.  
Так же сборка позволяет разбить HTML файл на отдельные блоки(header, footer, nav...)

---

### Шаги для запуска сборки

1. Клонируем репозиторий себе на компьютер
2. В консоли строке прописываем `npm init -y`
3. Прописываем в консоли `npm -i` и ждем пока установятся все необходимые зависимости
4. Прописываем в консоли `gulp` и запускаем сборку
5. Для публикации на Github-Pages, в консли вводим `npm run deploy`

### Список зависимостей

1. **gulp**

   > Собственно сам сборщик

2. **browser-sync**
   > Предназначен для автоматического обновления страницы при внесении изменений в код, а так же для запуска браузера при старте сборки
3. **del**
   > Выполняет очистку папки **dist** при запуске сборки
4. **sass**
   > ядро препроцессора SASS
5. **gulp-sass**
   > Плагин для компиляции SASS/SCSS в CSS код, понятный браузеру
6. **gulp-file-include**
   > Позволяет разбивать HTML документ на отдельные блоки и переиспользовать их на разных страницах сайта
7. **gulp-imagemin**
   > Автоматически оптимизирует изображения форматов jpg и png
8. **gulp-replace**
   > В данной сборке добавлен для замены путей изображений в scss при использовании свойств background-image и background
9. **gulp-sourcemaps**
   > Плагин для генерации _css.map_ файлов
10. **gulp-autoprefixer**
    > Плагин для автоматической расстановки префиксов
11. **gulp-rename**
    > Простой плагин для переименования файлов. В сборке добавляет .min к файлу css

### Переменные

#### Переменные по умолчанию для шрифтов

`$font-family: "Open Sans";` - Шрифт по умолчанию замените на тот что у вас

`$font-bold: 700 24px/140% $font-family;` - Это сокращенная запись свойств для шрифтов, меняйте размер и высоту строки.

**Расшифровка**

- 700 - font-weight
- 24 - font-size
- 140% - line-height
